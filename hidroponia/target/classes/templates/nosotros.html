Información del Proyecto

Este proyecto . Consistió en crear un sistema hidropónico vertical que nos tomó 2 semanas construir. Una de nuestras primeras dificultades fue diseñarlo correctamente; antes de iniciar la construcción hicimos un boceto para decidir la estructura final del sistema.

Después empezamos con la germinación de las plantas, aprendiendo cómo germinarlas sin tierra para lograr un crecimiento eficiente. También comenzamos el desarrollo del programa, donde incluimos información completa sobre qué es un sistema hidropónico, sus características, beneficios y usos.

Añadimos los materiales utilizados, el proceso paso a paso de germinación y el trasplante. Incluimos imágenes del desarrollo y de los productos finales que obtuvimos. Además, elaboramos una tabla sobre cómo debe configurarse correctamente un sistema hidropónico sin tierra.

Finalmente, para la parte de Módulo II, creamos un sistema de registro de plantas que nos permite almacenar en una base de datos el nombre de la planta, tipo, fecha de registro y nivel de nutrientes. También agregamos un formulario para que personas externas puedan solicitar información o enviar dudas al encargado del proyecto.
